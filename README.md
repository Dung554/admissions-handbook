## INTRODUCTION
The **360 Book System** is a digital platform that replaces traditional printed guidebooks and fragmented online sources used by students and parents to gather university admission information.  
This system centralizes and streamlines the process of exploring subject combinations, university programs, admission schedules, and personalized career guidance â€” all in one web-based solution.  

By providing a comprehensive and interactive environment, the platform empowers users to easily access, compare, and manage admission-related data while improving the overall admission experience for both students and educational institutions.

---

## USER ROLES

### 1. Guest
- View university admission scores and schedules.  
- Search for universities by **major name**, **subject groups**, or **admission scores**.  
- Filter and explore suitable majors based on interests or academic strengths.  
- Access educational news, articles, and updates.  
- Log in using Google or a system account to unlock additional features.

### 2. User (Student/Parent)
- Calculate admission scores using built-in tools.  
- Receive personalized **notifications** about favorite majors or universities.  
- Comment on educational news articles and participate in discussions.  
- Communicate and exchange information in the **forum**.  
- Submit **feedback** or inquiries directly through the system.  
- Manage and edit personal **profiles**.

### 3. Staff
- Verify user-submitted information and maintain data accuracy.  
- Support users through **chat** and provide admission consultations.  
- Manage incoming feedback and coordinate with the admin team.  

### 4. Admin
- **News Management:** Review and publish educational news articles.  
- **University Management:** Add or update university, major, and subject group information.  
- **Feedback Management:** Receive, review, and resolve feedback from users.  
- **User Management:** Monitor and manage user activities and permissions.  

---

## MAJOR FEATURES

- **University Search:** Explore universities by major name, subject group, or personal scores.  
- **Admission Schedule:** Stay updated with official admission timelines.  
- **News Feed:** Read educational updates curated by the system admin.  
- **Score Calculator:** Calculate and predict potential admission outcomes.  
- **Forum:** Engage with other users and share admission experiences.  
- **Feedback System:** Submit opinions or issues for system improvement.  
- **Notification Service:** Receive alerts about favorite majors and universities.  
- **Profile Management:** Update personal details and manage preferences.  
- **Google Login:** Secure and quick access via Google account.  
- **Admin Tools:** Review news, manage universities, and handle feedback efficiently.  
- **Chat Service:** Real-time communication for consultation and support.

## TECHNOLOGIES USED

**Frontend:** ReactJS, HTML, CSS  
**Backend:** Spring Boot (Java)  
**Database:** MySQL  
**Authentication:** Google OAuth 2.0, System Account  
**Architecture:** Microservices / API Gateway Integration  
**Deployment:** Docker, GitHub Actions (CI/CD), VPS Hosting
